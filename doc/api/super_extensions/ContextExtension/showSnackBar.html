Extension method on BuildContext to show a snackbar. It takes snackBar as the parameter of type SnackBar. If you are creating your own SnackBar, it's suggested to extend the class with SnackBar. Example to create your own SnackBar class:

class CustomSnackBar extends SnackBar {
CustomSnackBar({
 super.key,
 required this.message,
}) : super(
       content: Text(message),
        backgroundColor: Colors.black,
     );
 final String message;
}
